Пояснение как работать с менеджером для разработчиков. 

Для начала пару слов про создание репозитория тут всё просто.

в корне любого сайт вы должны создать файл ".dd" Пример наполнения смотреть > .dd
Где :example1:, :example2: и :example3: имена пакетов по которым происходт установка, а url это соответственно ссылка на файл.

Как вы уже поняли пакет представляет собой zip архив в корне которого лежит main.sh Запускаемый при установке дальше вы можете сделать что угодно создать команду которая выполнится один раз или установить свою программу всё в ваших руках.

Если вам необходимо установить deb пакет воспользуйтесь sudo dpkg -i <Название deb файла> и положить его вместе с исполняемым файлом
