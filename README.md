Lum - пакетный менеджер решивший отказаться от стандартных форм пакетов и дать разработчикам больше свободы.

Он крайне прост в использовании. Для установки пакета введите:<br>
sudo lum install [Имя пакета]<br>
Для удаления: <br>
sudo lum remove [Имя пакета]<br>
Для обновления локальной базы пакетов:<br>
sudo lum sync [Ссылка на сторонний репозиторий если не указывать будет использован наш репозиторий]<br>

Установка:
(Перед началом установки пакетного менеджера установите python3 и pip, а также git для возможности склонировать репозиторий.)

Клонировать репозиторий, перейти в директорию клонированного репозитория выполнить файл install.sh.
Так же можете использовать команду:

~~~
sudo curl -o /usr/bin/lum https://raw.githubusercontent.com/110000110000/lum/main/lum && sudo chmod +x /usr/bin/lum && sudo mkdir /lumRm && sudo mkdir /lum && pip install requests && pip install progressbar
~~~
