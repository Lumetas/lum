Lum - пакетный менеджер решивший попытаться от стандартных форм пакетов и дать разработчикам больше свободы.

Он крайне прост в использовании. Вы запускаете файл install.sh И ожидаете завершения установки после завершения вы можете спокойно его использовать.
Для установки пакета введите 

lum <Имя пакета> <Доменное имя стороннего репозитория> <ключ -s если сторонний репозиторий использует ssl сертификат.>
