Lum - пакетный менеджер решивший отказаться от стандартных форм пакетов и дать разработчикам больше свободы.

Он крайне прост в использовании. Для установки пакета введите:

lum <Имя пакета> <Доменное имя стороннего репозитория> <ключ -s если сторонний репозиторий использует ssl сертификат.(Работа с ssl не гарантирована, даже больше, скорее всего не возможна)> 
Например:

lum test

lum test test.domen.ru -s

В случае использования стандартного репозитория и стороннего репозитория с ssl соответственно.

Установка:
Перед началом установки пакетного менеджера установите python3 и pip, а также git для возможности склонировать репозиторий.

Клонировать репозиторий, перейти в директорию клонированного репозитория выполнить файл install.sh.
